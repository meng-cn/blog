Blog
==== 

New post
------------
```
$ hexo new [layout] <title>
```

New page
------------
```
$ cd site-dir
$ hexo new page <page-name>
```

Config
------------
- project config: _config.yml
- theme config: themes/theme-name/_config.yml

Test
------------
```
$ hexo server
```
